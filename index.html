A toolbox distro for young developers

cs.edu is a reboot of programming-oriented distros, providing either a full-fledged debian-based, debian-compatible distro, or a set of scripts to install most of the programming tools computer science students need.

What is it ?

cs.edu is still alpha software. It plans to brings users two things: a set of scripts/lists of interesting software ; an iso, ready to install.

History

Its name came from cs.edu Slax, a distro that stopped in 2007. A dire need for a ready-to program distribution to install on computers in the University of Nantes started the project.
